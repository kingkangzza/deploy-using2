CBR: 조출생률 (Crude birth rate) 연앙인구 대비 출생아수 변화

- SD: 시도 이름 (string)
- SGG: 시군구 이름 (string)
- NAME_AD: 행정동 이름 (string)
- CODE: 행정동 코드 (int64)
- CODE_AD: 행정기관 코드 (int64)
- YEAR: 해당 데이터 연도 (int64)
- B_ALL: 전체 출생아 수 (float64)
- B_M: 남아 출생아 수 (float64)
- B_F: 여아 출생아 수 (float64)
- PM_ALL: 전체 연앙인구 수 (float64)
- PM_M: 남자 연앙인구 수 (float64)
- PM_F: 여자 연앙인구 수 (float64)
- CBR: 조출생률 (float64)
