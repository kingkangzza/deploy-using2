FI: 재정자립도 CSV

- SD: 시도 이름 (string)
- SGG: 시군구 이름 (string)
- CODE: 행정동 코드 (int64)
- CODE_AD: 행정기관 코드 (int64)
- FI: 재정자립도(float64)
- YEAR: 해당 데이터 연도 (int64)
- INTEREST: 지방연 관심지역(int32) (1: 인구감소지역/ 2: 관심지역)
