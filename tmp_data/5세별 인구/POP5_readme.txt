POP5: 읍면동 단위의 5세별 인구 수 

- SD: 시도 이름 (string)
- SGG: 시군구 이름 (string)
- NAME_AD: 행정동 이름 (string)
- CODE: 행정동 코드 (int64)
- CODE_AD: 행정기관 코드 (int64)
- YEAR: 해당 데이터 연도 (int64)
- ALL: 해당 연도 총 인구수
- '0_4', '5_9', '10_14', '15_19', '20_24', '25_29', '30_34', '35_39', '40_44', '45_49',
       '50_54', '55_59', '60_64', '65_69', '70_74', '75_79', '80_84', '85_89',
       '90_94', '95_99', '100_', : 5세 별 인구 수 (float32)
